# T1 统计方法链对照表

选题提示中的方法链：

```text
MLE 点估计
→ Poisson 置信区间（Wald / Score / Exact）
→ Block Bootstrap（BCa 方法）
→ 两独立 Poisson 条件检验
→ 分层估计（按装备类型 / 时间段）
```

## 当前版本与选题提示的差距

当前版本不是完全严格照搬方法链，而是根据可获得数据做了可行实现：

1. Kaggle / WarSpotting 逐条数据只有俄罗斯损失，适合做俄罗斯时间序列。
2. Oryx 双方页面只有汇总数据，适合做双方总体比较和装备类型分层比较。
3. 因为缺少乌方逐日或逐周数据，严格的双方 Block Bootstrap 和按时间段分层估计暂时做不了。
4. 因为当前 Python 环境没有 `scipy`，Poisson Exact 区间暂时没有计算；后续可安装 `scipy` 后补充。

## 建议论文表述

推荐写法：

“本文以教材中的泊松分布、点估计、最大似然估计和区间估计为主线。按照选题提示，进一步尝试 Bootstrap 和分层估计。受限于可获得数据，本文对 Oryx 双方汇总数据采用参数 Bootstrap 作为稳健性分析；严格的 Block Bootstrap 和 BCa 区间需要双方逐日损失记录，本文将其作为后续扩展。”

这样写比较诚实，也能说明我们理解了题目要求。

## 后续补齐方案

若要最大限度贴合选题提示，下一步可以做：

1. 如果找到乌方逐日损失数据，改用双方每日或每周序列做 Block Bootstrap。
2. 在获得双方时间序列后，实现 BCa Bootstrap 区间。
3. 按年份或季度做双方分层 Poisson 估计和检验。
