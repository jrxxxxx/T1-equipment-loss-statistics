# T1 研究方案

## 题目

俄乌装备损失率“公平比较”：从估计偏误到区间估计

## 一、研究目标

本项目基于 Oryx / WarSpotting 等开源影像验证装备损失数据，比较俄乌双方装备损失情况，并使用数理统计方法给出点估计、区间估计和显著性检验结果。

核心问题：

1. 如何估计俄乌双方装备损失强度或损失率？
2. 双方装备损失是否存在统计显著差异？
3. 不同装备类型、不同时间阶段下，损失差异是否一致？
4. 开源影像验证数据可能存在怎样的观测偏误？这种偏误会如何影响结论？

## 三、核心变量定义

### 1. 损失数量

设某一时间窗口内，俄罗斯装备损失数量为：

`X_R`

乌克兰装备损失数量为：

`X_U`

### 2. 损失占比

俄罗斯损失占比：

`p_R = X_R / (X_R + X_U)`

乌克兰损失占比：

`p_U = X_U / (X_R + X_U)`

### 3. 损失强度

如果以天或周为单位统计，设观测时间长度为 `T`，则：

`lambda_R = X_R / T`

`lambda_U = X_U / T`

其中 `lambda_R` 和 `lambda_U` 表示单位时间内的平均装备损失强度。

### 4. 损失强度比

`RR = lambda_R / lambda_U`

若 `RR > 1`，表示俄罗斯单位时间装备损失强度更高；若 `RR < 1`，表示乌克兰更高。

## 四、统计方法路线

### 1. 描述性统计

完成以下图表：

- 双方总损失数量柱状图
- 按装备类型分组的损失数量柱状图
- 每日或每周装备损失时间序列
- 不同损失状态的构成图
- 主要装备类型的损失占比对比图

### 2. 点估计

将单位时间装备损失数量近似看作 Poisson 过程：

`X_R ~ Poisson(lambda_R T)`

`X_U ~ Poisson(lambda_U T)`

最大似然估计为：

`lambda_hat_R = X_R / T`

`lambda_hat_U = X_U / T`

进一步估计损失强度比：

`RR_hat = lambda_hat_R / lambda_hat_U`

### 3. Poisson 置信区间

对双方损失强度分别构造置信区间：

- Wald 近似区间
- Score 区间
- Exact 区间

比较不同区间方法的差异，并解释为什么小样本或稀有装备类型下 Exact 区间更稳健。

### 4. Bootstrap 区间估计

考虑装备损失数据可能存在时间相关性，因此使用 Block Bootstrap：

1. 将每日数据按周或固定长度区块分组
2. 有放回抽取区块
3. 重复计算 `lambda_R`、`lambda_U` 和 `RR`
4. 得到 Bootstrap 分布
5. 构造百分位区间或 BCa 区间

### 5. 假设检验

检验双方损失强度是否相等：

`H0: lambda_R = lambda_U`

`H1: lambda_R != lambda_U`

可使用：

- 两独立 Poisson 率检验
- 条件精确检验
- Bootstrap 检验

输出：

- 检验统计量
- p 值
- 是否拒绝原假设
- 统计意义与现实意义解释

### 6. 分层分析

按以下维度重复估计和检验：

- 装备类型：坦克、装甲车、火炮、飞机、无人机等
- 时间阶段：2022、2023、2024、2025，或按季度/半年划分
- 损失状态：destroyed / damaged / abandoned / captured

分层分析的目标是判断：

- 总体差异是否由某几类装备主导？
- 不同阶段的损失格局是否发生变化？
- 某些装备类型的差异是否更显著？

## 五、估计偏误讨论

Oryx / WarSpotting 是“被影像验证的损失”，不是全部真实损失。潜在偏误包括：

1. 观测概率不同：前线、城市、开阔地带的损失更容易被记录。
2. 双方宣传与信息环境不同：不同阵营上传影像的积极性不同。
3. 装备类型差异：大型装备更容易被识别，小型装备更容易漏记。
4. 时间差异：早期和后期的数据完整性可能不同。

因此本文结论应表述为：

“在开源影像验证数据所反映的可观测损失中，双方损失强度存在/不存在统计显著差异。”

而不是直接断言真实全部损失。

可选扩展：

- 做敏感性分析：假设某一方观测概率为另一方的 0.8、1.0、1.2 倍，观察结论是否稳定。
- 按装备类型加权，降低不同装备被观测概率差异带来的影响。
